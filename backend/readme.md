# Toma Ramos

Toma Ramos is a personal portfolio project focused on building a course enrollment system.  
The backend is implemented using Express, PostgreSQL, and Docker. The frontend will be developed using Angular.

This repository is intended to demonstrate backend architecture, database design, and API development practices.

---

## Tech Stack

### Backend
- Node.js
- Express
- PostgreSQL
- Docker & Docker Compose
- JWT-based authentication
- Password hashing for user registration

### Frontend
- Angular (planned)

---

## Project Status

The project is currently in an early development stage.

At this point, the backend foundation is implemented, including:
- Initial PostgreSQL database schema
- Core API endpoint constraints and validations
- User registration with secure password hashing
- Dockerized local development environment

Frontend development has not started yet and will be addressed in future iterations.
